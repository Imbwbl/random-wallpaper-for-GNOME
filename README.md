# random-wallpaper-for-GNOME
this is a custom shell script who download image from wallhaven.cc and apply it to background of GNOME
you can change the collection id in the wall shell script
you need to add the .desktop to /usr/share/applications and add desktop to /usr/local/bin
open gnome tweaks and add the wall app in startup app if you don't see it try to alt+f2 and type r for restart GNOME
